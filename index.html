Features

Real data. Cosmic language.

Every line of the horoscope is driven by actual numbers from the CoinStats API. The absurdity is real, but so is the data.

📡

Live market data

Pulls real-time prices, 1h/1d/1w/1m % changes, BTC dominance, and volume from the CoinStats API on every run.

🎲

Deterministic per day

Seeded by today's date + current prices. Same data = same reading. Fresh prophecy every morning, stable all day.

🪙

Any coins

Pass any symbols you hold: BTC ETH SOL DOGE. Falls back to the top 10 by market cap.

☿

Mercury's Curse

Your worst daily performer gets called out with appropriately dramatic astrological commentary. Every. Time.

♀

Venus's Blessing

Your best weekly performer receives celestial praise. The stars have a favorite, and today it might be you.

📤

Three output modes

Rich terminal with colors and panels, --share for Discord/Slack, or --json for piping.

Get started

Up in 60 seconds

Python 3.9+, two dependencies, one environment variable.

Install
$ git clone https://github.com/MarthaPetrosyanCoinstats/crypto-horoscope $ cd crypto-horoscope && pip install -r requirements.txt $ export COINSTATS_API_KEY=your_key_here # free at openapi.coinstats.app
Usage
# Read for your holdings $ python3 horoscope.py BTC ETH SOL DOGE # Default: top 10 coins by market cap $ python3 horoscope.py # Shareable text for Discord / Slack / Twitter $ python3 horoscope.py --share BTC ETH SOL # Structured JSON output $ python3 horoscope.py --json BTC ETH | jq '.coins[].reading'

How it works

Astrology, but make it algorithmic

The readings sound like a psychic wrote them, but they're fully driven by real market numbers.

1

Fetch live data

Calls the CoinStats API for the top 500 coins and global market stats. Filters to your requested symbols, deduplicating by market cap rank so Binance-Peg tokens don't sneak in.

2

Seed the oracle

Combines today's date with a hash of current prices to seed Python's random.Random. Same market conditions → same reading. Different day → different reading.

3

Map data to tone

Price changes classify each coin as moon_shot / bullish / neutral / bearish / crater. Monthly trend adds context. BTC dominance and volume shifts drive the market-wide narrative.

4

Select & render

Template banks keyed on tone + magnitude produce the prose. The rich library renders colored panels in the terminal, or you get clean plain text for sharing.

What you get

Actually useful.
Accidentally funny.

📡

Real data, not vibes

Every reading pulls live prices, 1h / 1d / 1w / 1m changes, BTC dominance, and volume from the CoinStats API. The absurdity is real. The numbers are too.

🗓

Fresh every morning

Seeded by today's date + current prices. The reading is consistent all day, different tomorrow — like a real horoscope, except the data is verifiable.

🪙

Any coins you hold

Pass your actual holdings: BTC ETH SOL DOGE. Falls back to the top 10 by market cap if you'd rather not decide.

📤

Three output modes

Rich terminal with colors for yourself, --share for Discord and Slack, or --json for piping into whatever you're building.

☿

Mercury's Curse

Your worst daily performer gets called out personally, with appropriately dramatic astrological commentary. Mercury shows up every single day.

♀

Venus's Blessing

Your best weekly performer receives celestial praise. The cosmos has favorites. Some weeks, it might even be you.

Get started

Up in 60 seconds

Python 3.9+, two packages, one environment variable. That's genuinely all of it.

Install
$ git clone https://github.com/MarthaPetrosyanCoinstats/crypto-horoscope $ cd crypto-horoscope && pip install -r requirements.txt $ export COINSTATS_API_KEY=your_key_here # free at openapi.coinstats.app
Usage
# Your holdings, cosmically interpreted $ python3 horoscope.py BTC ETH SOL DOGE # Paste-ready text for Discord / Slack $ python3 horoscope.py --share BTC ETH SOL # Raw JSON for piping or automation $ python3 horoscope.py --json BTC ETH | jq '.coins[].voices.oracle'
